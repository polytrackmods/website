PolyModLoader

About

PolyModLoader is a Mod Loader for the browser game PolyTrack by Kodub, developed by Orangy, Lockheed SR-71 Blackbird, CRJakob, Dorachad and SpideyZac.

Features

PolyModLoader offers many features for Players and Developers to help make playing with and creating mods easier and more accessible

For Players

  • Using multiple mods at once
  • Play on both App and Browser version
  • Mobile support
  • Audomatic mod updates
  • Mod settings

For Developers

  • Mixins
  • APIs for making mods without ever needing to mixin code:
    • Mod Settings
    • Keybinds
    • Popups
    • Custom Blocks
